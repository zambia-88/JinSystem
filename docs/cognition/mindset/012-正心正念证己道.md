# 正心正念证己道

1、永远不要问原因:话越少，精力越集中，气场也更大；
2、允许一切发生:不要对抗，而是坦然接受，有能力就去改变，反之…
3、不要过度窥视未来:人性禁不住考验，与其无妄担忧未来，不如守好自己的本心，做好现在；
4、管好自己的情绪，守住心:少说话，少插嘴，少共情，少怜悯；
5、不过度同情任何人:你同情谁，潜意识就背负了谁的命运，看似伟大，实则愚蠢。人性都是经不起考验的，以自我为中心的人为 99.99%；
6、接受所有的事情发生:相信因果，看似偶然的发生，一定有其宿命中的必然；
7、独立独往是正道:不盲从流行或权威，不依赖团队和他人，可以自己判断与决定，毕竟人生是自己的！
