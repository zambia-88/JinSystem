# 维度提升

提升维度的关键不在于解决问题，
因为有一些问题，你站在同一平面，它根本就是无解的，关键在于让这个问题不再成为问题！
比如你在迷宫中绕来绕去，肯定到处都是墙，
但当你站在迷宫的上头，看到的到处都是路，
我们无法用制造问题时的统一水平思维来解决问题！
升维并不是去解决问题本身，而是进入一个更高的维度让问题直接消失…
