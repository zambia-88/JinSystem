# 意志恒心
To be a better man

很多时候，长期主义的写照是:
成功不在于你做了什么，
而在于你没做什么；
时间是创新者的朋友，
是守城者的敌人；
寓沉雄于静穆，
藏锋芒于深思；
所谓:
初有决定不移之志；
中有勇猛精进之心；
末有坚贞永固之力！
