# 人性成长

做事→做人→做局→破局→控局→掌局
做事→立名→混圈→人脉→助人→达己
1. 做事的人遇到做人的人皆被降服…
2. 做人的人遇到做局的人，就算你把人做的再好都只是做局人的一颗棋子…
3. 做局的人遇到破局的人，看透人性与规律，掌握规律，顺势而为…
4. 正所谓，做事有余，出言有度，下水要懂水性，做事要懂分寸…
5. 富在术数，不在劳身，利在局势，不在力耕！
