# 写给犹豫不决不愿意变化的自己

1、看见自己拥有的而不是困在自己想要的：不要把自己拥有的东西当做理所当然，它是阻碍幸福的杀手，真正的幸福是你能感受到自己已经拥有了什么；
2、对错之外，还有成长：不要像小孩子那样争对错，成年人的世界里，对错是孩子的游戏，得失才是成年人的考题，如果争执能够带来理解，那就据理力争，如果沉默能够保护重要的人，那就一笑而过。真正的成熟，不是放弃立场，而是知道什么时候放手，什么时候坚持？
3、学会放下，睡个好觉：天塌下来都要睡好觉，这一生，挣钱只是游戏，或者才是胜利，快乐是真谛，健康是目的，钱永远也挣不完，但一定会耗尽你自己；
4、别为没得到的皱眉，多看看身边的暖，别为对错钻牛角尖，多想想心里的情，别把事都扛肩上，多给自己留轻松，生活安稳，家庭幸福，每日睡香甜，比什么都强。
