# 人在前，事在后

冯唐说：“越是社会底层的人，社交能力越差，懂人情世故的人不多，单纯以为把事情做好，就能弥补人情事故上的缺陷。 殊不知，你越是往上爬，越会认识到，大部分社会上层的人，其实是在研究人，而不是在琢磨事。” 
💪(≧ω≦)那就好好研究人吧，在今后的日子里！
