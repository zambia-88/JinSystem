# 认知水平

认知水平越低的人，动物本性就越强，他就越不识好歹，你表现出来的善良，在他看来就是好欺负，他就总想着占那点便宜。他根本就不明白，你善良背后的可贵之处，他们只会不知不觉的把你当成弱者来看待，欺负你的心理，就像欺负一只蚂蚁一样。正如俗话说:小人畏威不畏德！
