# 关于决断力

难走的路，不是叫你停下来的
晨跑本就消耗体能，遇到难跑的路，停下来休息，在逻辑上完全成立。那一刻，我清楚地知道：这是一个“足够合理”的接口。
但我还是选择拒绝执行。因为我意识到，停下来确实可以暂时轻松，却会拉长整体时间成本，更重要的是—它不会让我的决断力变强。
我选择快速跑过去。在更快地节奏里逼迫自己判断、找点、落脚，当地在动态中完成选择，穿过那一条路，光自然会出现。
我越来越确认：这种能力，不会自然增长，只能通过有意识的训练获得。而这类训练，必须发生在你“完全有理由停下来”的时候。
