# 职场能力

千万，不要过度透支自己的能力:
能力透支的本质是时间维度上的失衡。
真正成熟的职场人，都懂得在"全力以赴"与"留白生长"间找到平衡点。
守住能力边界，方能细水长流；
持续赋能更新，才是永续之道。
