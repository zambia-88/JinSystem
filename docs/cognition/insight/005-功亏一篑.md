# 功亏一篑

《第99次跃起》

当你数到第99次跃起时
脚尖已提前勾勒出胜利的弧线
可就在那本该完美的终章
绳柄突然亲吻脚踝
在离地3厘米处
完成了一次对骄傲的精准绞杀

我们总在潜意识里提前鸣响庆贺的钟声
却让最后一记心跳
成为溃败的号角
那些被风吹散的99次坚持
都在提醒着：
真正的抵达
永远在触手可及的下一次振臂

「胜利不会缺席，
但会惩罚提前退场的心跳」
