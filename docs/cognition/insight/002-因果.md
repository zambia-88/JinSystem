# 因果

人们相信美好，却又选择性的忽略了事物发展相关的因果必然性。如果美好是左，因果是右，那人生就是一个不断被左右的过程。我们负重前行，只为在因果的必然性中创造美好。

左手美好，右手因果。人生，一场于必然之右，求索美好之左的执着。我们被“左右”，亦在“左右”中创造。
