# 插入多张
不规则图片

新建画布，插入文本框，输入短横线，选中文本框，在形状格式中，文本效果，点击转换，选择图形形状（朝鲜鼓），文本选项中调整间距为紧密，度量值3磅，插入矩形，先选中文本框，再选中矩形，形状格式选拆分，复制粘贴图片，C太热了+X剪切图片，右键设置形状格式，纹理或图片填充，剪切板，以此操作
