# 谷歌编程（Google Antigravity）
更牛逼的3D可视化生成及演示

先在电脑桌面上新建一个名为 3D_Project 的纯英文文件夹，把 .xlsx 放进去，再通过 Open Folder 打开这个特定的文件夹。这样可以避免扫描整个磁盘带来的系统沉重感。
使用方法：利用Gemini设计提示词，上传Google Antigravity系统内生成的界面截图，发到Gemini中进行解读并给出建议和提示词，再到Google Antigravity系统内进行任务生成，直到生成满意的任务为止
