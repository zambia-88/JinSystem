# 读典籍

一个很有雅兴的网站，能阅读许多经典古籍，学习前人智慧，感悟中华文化。

网站内可体验文白对照、交互式注释及导读工具，提供文章分节、人物、事件等丰富阅读辅助功能。以《孙子兵法》为例，不仅每句对照释义，还提供章节目录、关键人物介绍以及重要事件典故等，让人能更深刻地阅读。
