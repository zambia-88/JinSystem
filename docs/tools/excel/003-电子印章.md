# 电子印章

空白表格点插入形状，选择圆形，按住shift拖动鼠标拉出一个正圆形，鼠标右击设置为无填充颜色，边框设置为印章的边框颜色，粗细设置为3磅，插入艺术字，输入公司名称，点击效果，选择转换，选择上弯弧，拖动调整弧度适宜，设置字体印章颜色，插入对应图案，按住shift键，把颜色边框设置为印章色，选择所有形状，选择水平和垂直居中，框选全部，点击组合就行
