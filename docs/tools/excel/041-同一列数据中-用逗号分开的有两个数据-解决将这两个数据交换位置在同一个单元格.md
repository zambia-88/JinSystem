# 同一列数据中，用逗号分开的有两个数据。解决将这两个数据交换位置在同一个单元格。

在B列单元格输入公式，向下填充即可：=MID(A1,FIND(",",A1)+1,100)&","&LEFT(A1,FIND(",",A1)-1)
