# 图片批量整齐嵌入单元格

先把图片全部导入，全选后右键，找到大小和属性，关闭锁定纵横比，高度、宽度调整为单元格大小，点击最后一张图片，移动到最后一个边框单元格，按住Alt键吸附边框，最后再把图片全选，找到对齐，点击左对齐，再点击对齐，好到纵向分布即可
