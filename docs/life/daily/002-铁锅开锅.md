# 铁锅开锅

1、用百洁布和洗洁精将锅洗干净；
2、大火上灶把锅烤至微蓝烤透，等自然冷却，用磨刀石钝化锅边；
3、用钢丝球炒粗盐，洗洁精水洗净；
4、开火用猪皮涂抹均匀，反面一样，再来点食用油，加入一点青菜翻炒捞出洗净即可；
