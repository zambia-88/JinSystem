# 饺子皮葱油饼

用小葱炸葱油，均匀刷在一块饺子皮上，再盖上一块饺子皮，继续刷小葱和葱油，再盖上一块饺子皮，用器具擀平，然后卷起来，竖向用手掌用力压平成饼胚，保鲜膜封住醒面十分钟放在油锅里两面煎至金黄即可
