# 鲈鱼蒸蛋

鲈鱼用葱姜水去腥后洗干净再用开水烫一下，3个鸡蛋加开水（1:1.5），加盐搅散，撇去浮沫，淋上鸡蛋液，两勺蒜蓉酱，，扣盖上汽蒸10分钟关火焖2分钟，葱花用热油呛炒淋入盘里，蒸鱼豉油用热锅呛一下即可
