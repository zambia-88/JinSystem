# 矿产开发部

<div class="reading-prose">

矿产开发部（MDD）的职责是促进矿产资源开发，确保所有采矿活动均按照《矿产和矿产开发法》及其附属立法的规定进行，确保矿产资源开发政策的实施。

MDD 的职能包括颁发和管理非采矿权（矿产贸易许可证、矿产进出口许可证和淘金证书），监督采矿和矿物加工作业，以确保其发展符合批准的运营计划并符合 2015 年《矿山和矿产开发法》第 11 条的规定。该部门还负责收集、汇编和传播矿产生产统计数据，并进行矿坑调查、设立信标和颁发挂钩证书，以及划定和调查采矿权。

技术部门结构

矿产开发部由以下部门组成：–

矿山开发

运营

矿产经济学

矿产开发科

矿山开发科（小型矿业开发）专注于小型采矿和矿物加工业务的发展。本科分为以下两个 (2) 个单元：

勘察单位

该部门负责：

采矿权区域划定与调查；

安装灯塔并颁发锚定证书；

矿坑勘测；以及

维护和更新所有采矿权区域的调查记录。

地区矿业局单位

该部门负责为小规模采矿者和矿物加工者提供现场技术服务。该部门共有六 (6) 个区域采矿局 (RMB)。区域采矿局位于以下城镇；

索尔韦齐 (西北省)

基特韦 (铜带省)

曼萨（卢阿普拉/北方省）

姆库希 (中央省)

奇帕塔 (东部/穆钦加省)

乔玛（南部/西部省）

行动组

运营部门（大型采矿业务）专注于评估拟议的大型采矿企业并监控大型运营以确保合规。本部门分为以下两个 (2) 个单元：

行动组

该部门负责监督和检查大型采矿作业，以确保遵守批准的采矿和矿物加工计划，并确保无浪费的采矿实践。

项目评估组

该部门负责通过可行性研究报告评估采矿项目，以确定其可行性并确保其按照已批准的运营计划进行开发。该部门还负责评估和监督采矿和矿物加工项目，以确保项目进入生产阶段。

矿产经济科

矿产经济部门专注于非采矿权的颁发和管理、矿产统计数据的收集、编制和处理，以及价值链各个环节的矿产生产的监测和核查。本部门分为两 (2) 个单元；

矿产经济部门

矿产经济部门负责：

收集、汇编和处理采矿统计数据（矿产生产、矿产出口和销售、金属/矿产价格、采矿投资和劳动力统计数据），以监测绩效并向最终用户提供数据；

办理矿产进出口许可证、矿产贸易许可证、淘金证。

矿产价值链部门

该单位负责监控矿物从矿场开采点到地磅运输，以及最终在边境口岸和机场出口的全过程。
