1. **框架优先** — 法律、土地、税收与行业政策先厘清  
2. **周期思维** — 尊重大宗商品的长周期属性  
3. **长期跟踪** — 同一区域与主题持续迭代  
4. **公开边界** — 仅发布可公开分享的研究与整理资料
